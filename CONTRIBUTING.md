# Contributing

Thanks for wanting to improve GideonMail.

## Reporting bugs

Open a [Bug Report](https://github.com/gbroeckling/gideonmail/issues/new?template=bug_report.yml) with:

- OS and version
- Email server type (ISPConfig, Dovecot, etc.)
- Steps to reproduce
- Error messages or screenshots

## Requesting features

Open a [Feature Request](https://github.com/gbroeckling/gideonmail/issues/new?template=feature_request.yml)
describing your use case and motivation.

## How to submit

1. Fork the repo
2. Create a branch: `feature/your-change`
3. `npm install && npm start` to test
4. Commit with clear messages
5. Open a PR describing what changed and how to test it
